# Meeting Preparation

When preparing for a meeting, follow this workflow:

## 1. Gather Meeting Details
- Use the **WorkIQ** tool (`ask_work_iq`) to pull upcoming meeting details: title, time, attendees, and any attached agenda or documents
- If a specific meeting is mentioned, search for it by name or attendee
- If no specific meeting is mentioned, pull the next upcoming meeting or today's schedule

## 2. Draft the Agenda
Structure the agenda as:
- **Meeting title** and date/time
- **Attendees** (with roles if known)
- **Objective** — one sentence on what this meeting should accomplish
- **Discussion items** — numbered list with time allocations
- **Action items from last meeting** (if this is a recurring meeting, search for prior notes)
- **Parking lot** — space for off-topic items

## 3. Create Talking Points
For each discussion item:
- Draft 2-3 bullet points {{YOUR_NAME}} can reference
- Include relevant data points, metrics, or status updates
- Flag any decisions that need to be made
- Note any pre-reads or documents to reference (search OneDrive if relevant)

## 4. Identify Risks & Prep Needs
- Flag if any key attendees are missing
- Note if there are conflicting meetings
- Suggest documents to review beforehand
- Highlight any open action items from previous related meetings

## Output Format
Save meeting prep as a markdown file in the active project folder (or offer to save to `CopilotWorkspace\_shared-resources\templates\` if it's a reusable template).
